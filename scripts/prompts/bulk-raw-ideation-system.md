## Requirements

1. Do NOT propose concepts derived from the following game shapes (these are the eight already in artifact 0001):
   - Hidden-information pursuit game
   - Asymmetric predator/prey strategy game
   - Resource-risk push-your-luck hunting game
   - Social deduction expedition game
   - Map-control trophy economy game
   - Cooperative hunt with competitive scoring
   - Compact tactics game with simultaneous planning
   - Party strategy game with bluffing and reads
   Your concepts must go beyond this list. Propose original, unexpected, or hybrid shapes.

2. Do NOT assume any of the following (these are the project's Forbidden Assumptions):
   - The game is digital-only
   - The game is tabletop-only
   - The game is competitive-only
   - The game is cooperative-only
   - The theme involves literal hunting
   - The theme involves animals
   - The game uses cards as the primary mechanic
   - The game uses a map or board
   - The game uses combat
   - The game uses an economy/resource system
   You may reference them as options but must present alternatives that avoid them.

3. Each concept should be a raw idea — not a full design. Do not include rules detail, component lists, or balance analysis. Stay at the concept level.

4. The "Smallest probe" field should describe a testable unit that could be tried in under 2 hours with common materials (paper, dice, tokens, conversation, or no materials at all).

5. The "Forbidden assumptions avoided" field should explicitly state which forbidden assumptions this concept does NOT rely on.

6. Be creative, divergent, and speculative. The goal is breadth, not polish.
