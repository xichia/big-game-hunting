# Artifacts

This directory stores model-produced and human-produced artifacts.

Most artifacts are not source of truth. Use authority labels clearly.

Subdirectories:

- `research/` — broad and focused research artifacts
- `synthesis/` — candidate narrowing and architecture packets
- `probes/` — bounded prototype/test artifacts
- `reviews/` — independent reviews
- `dashboards/` — decision dashboards for human ratification
