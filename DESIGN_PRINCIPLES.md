# Design Principles

Authority level: SOURCE OF TRUTH

These principles govern Big Game Hunting design decisions.

## Primary Principle

A mechanic advances only if it improves accessibility, mastery, replayability, or emotional payoff — and does not damage the source-of-truth vision.

## Core Principles

1. Easy to start, hard to master.
2. No complexity without payoff.
3. Beginners must have meaningful choices.
4. Experts must have room for skill expression.
5. The best strategy should depend on context.
6. The game should be teachable quickly.
7. Watching skilled play should be interesting.
8. Repeated play should reveal deeper structure.
9. Players should feel agency even when outcomes are uncertain.
10. Prototypes should test the smallest meaningful unit before full buildout.

## Evaluation Questions

Every major artifact should answer:

```text
Does this make the game easier to adopt?
Does this make the game harder to master?
Does this increase replayability?
Does this create memorable player moments?
Does this avoid unnecessary rules burden?
```

## Design Tension

Big Game Hunting should live in the tension between immediate graspability and long-term depth.

The goal is not a simple game. The goal is an accessible game with hidden depth.
